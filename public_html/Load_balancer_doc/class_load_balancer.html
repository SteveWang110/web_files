Detailed Description

The LoadBalancer class manages the request queue and the webservers. Depending on the amount of requests currently in the queue, the Load balancer may add or delete servers to balance the workload. Furthermore, it will make sure to only delete servers that are not currently processing. The load balancer also will give requests to servers that are ready to take in requests and keeps track of the total amount of requests processed.
